Data Types

Data types tell Python what kind of value something is. Here are the most common ones:

  • int: Whole numbers (e.g., 5, -3)
  • float: Decimal numbers (e.g., 3.14, -0.1)
  • str: Text (e.g., "hello")
  • bool: True or False values

Example

# age is an int
age = 21
# price is a float
price = 19.99
# name is a string
name = "Alice"
# is_student is a boolean
is_student = True
