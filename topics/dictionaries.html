Dictionaries

A dictionary stores data as key-value pairs. Think of it like a real dictionary: you look up a word (key) to get its definition (value).

Example

person = {"name": "Alice", "age": 21}
# prints 'Alice'
print(person["name"])

Why use dictionaries?

  • Quickly find values using keys
  • Organize related data together

Real-life Example: Grocery List

Let's see how you might use a dictionary in a real program. Imagine you want to keep track of grocery items and their prices. A dictionary makes this easy! 

# Step 1: Make a dictionary of groceries and prices
groceries = {"apples": 1.50,
             "bread": 2.00 ,
             "milk": 2.50   }

# Step 2: Pick an item you want to look up
item = "bread"

# Step 3: Get the price using the item as the key
price = groceries[item]

# Step 4: Print out the result
print(f"The price of {item} is ${price}")
  1. Create a dictionary with items as keys and prices as values.
  2. Choose the item you want to check.
  3. Use the item name (the key) to get its price from the dictionary.
  4. Print the result.

This is a quick way to look up prices for any item in your grocery list. Dictionaries help you organize and find information fast!
