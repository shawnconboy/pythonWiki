Text Files

This page explains how to work with text files in Python — reading, writing, and appending. It's written simply for beginners and follows ideas from Al Sweigart's book.

Step 1: Importing Modules

Sometimes we write:

from pathlib import Path

This means "from the pathlib module, import the Path class" — so we can just write Path() instead of pathlib.Path(). It makes code shorter.

Other times, we use the whole module name like this:

import os

This brings in everything in the os module. To use a function from it, you write os.function_name().

Step 2: Creating a New Text File

To create and write to a text file, use the open() function with 'w' mode (w = write):

file = open("hello.txt", "w")
file.write("Hello, world!\n")
file.write("This is a text file.\n")
file.close()

This makes a file named hello.txt and writes two lines to it. \n means "new line." Always call close() when you're done.

Step 3: Reading from a File

To read the contents back, open it in 'r' mode (r = read):

file = open("hello.txt", "r")
content = file.read()
print(content)
file.close()

Step 4: Appending to a File

If you want to add to the file without erasing it, use 'a' mode (a = append):

file = open("hello.txt", "a")
file.write("Another line.\n")
file.close()

Step 5: Using "with open(...) as file"

This is a cleaner way to open files. It closes the file automatically, even if there's an error:

with open("hello.txt", "r") as file:
    content = file.read()
    print(content)

Step-by-Step Program

This program will
  • Create a file
  • Write some text
  • Read it back
  • Append to it
  • Show final result

    • # Step 1: Write to file
with open("notes.txt", "w") as file:
    file.write("My first note.\n")
    file.write("Python is easy.\n")

# Step 2: Read file
with open("notes.txt", "r") as file:
    content = file.read()
    print("Current contents:\n", content)

# Step 3: Append to file
with open("notes.txt", "a") as file:
    file.write("Adding one more line.\n")

# Step 4: Read final file
with open("notes.txt", "r") as file:
    final = file.read()
    print("\nFinal contents:\n", final)

    Summary

    • open("filename.txt", "mode") is how you work with files. Modes are: "w" = write, "r" = read, "a" = append.
    • file.write("text") writes text. file.read() reads it all.
    • with open(...) as file is best practice because it auto-closes the file.
