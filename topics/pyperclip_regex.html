Pyperclip and Regex for Beginners

This page will show you, step by step, how to use pyperclip and re (regular expressions) in Python, just like in Al Sweigart's books.

Step 1: Importing Modules

First, you need to import the modules. pyperclip lets you copy and paste text to and from your clipboard. re lets you search for patterns in text.

import pyperclip  # For clipboard operations
import re         # For regular expressions (pattern matching)

Step 2: Copying and Pasting with Pyperclip

Copy text to the clipboard:

pyperclip.copy('Hello, world!')

Paste text from the clipboard:

text = pyperclip.paste()
print(text)  # Output: Hello, world!

Step 3: Using Regex to Find Patterns

Suppose you want to find phone numbers in some text. You can use re for this:

text = "Call me at 415-555-1011 or 212-555-9999."
phone_regex = re.compile(r'\d{3}-\d{3}-\d{4}')
matches = phone_regex.findall(text)
print(matches)  # ['415-555-1011', '212-555-9999']

Step 4: Putting It All Together

Let's get text from the clipboard, find all phone numbers, and print them:

import pyperclip
import re

# Get text from clipboard
text = pyperclip.paste()

# Create regex pattern for phone numbers
phone_regex = re.compile(r'\d{3}-\d{3}-\d{4}')

# Find all matches
matches = phone_regex.findall(text)

# Print results
for match in matches:
    print('Found phone number:', match)

Summary

  • pyperclip lets you copy and paste text to and from the clipboard.
  • re lets you search for patterns in text.
  • You can combine them to search for things like phone numbers in clipboard text.
